# Genshin-danzhugame-Impact
Qt-based Genshin Impact Themed Brick Breaker Game
## Features
-4 different brick patterns(brick pattern auto-switch every 15 seconds)
real-time score tracking
## Controls
-<-/-> keys:move the paddle
Space key:Restart the game after game over
## How to Run
1.Clown this repoositity

2.Open the project in Qt Creator

3.Build and Run the project
